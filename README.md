# Datia Movie Challenge

👉 https://datiamovie.herokuapp.com/

_2 days Challenge consuming The Movie Database (TMDb)_

## Starting 🚀

_Clone the repository_

### Installation 🔧

_👉 npm install_

_👉 add config file with API and API KEYS_

_👉 npm start_

## Stack 📌

React js and Styled Components.
